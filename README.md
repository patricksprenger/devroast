# DevRoast

O **DevRoast** é uma aplicação web onde desenvolvedores podem colar seus trechos de código e receber uma avaliação "sincera" (um roast) baseada em boas práticas e padrões de desenvolvimento. O objetivo é aprender de forma divertida, expondo erros comuns e sugerindo melhorias com uma boa dose de sarcasmo.

## 🚀 Funcionalidades

- **Paste & Roast**: Cole seu código e deixe a IA analisar cada linha.
- **Leaderboard da Vergonha**: Veja os códigos mais "roastados" da internet, ranqueados por pontuação de qualidade (ou falta dela).
- **Code Highlighting**: Visualização elegante de código com suporte a temas modernos.
- **Roast Mode**: Escolha entre uma análise técnica séria ou o modo sarcasmo máximo.

## 🎓 Sobre o Projeto

Este projeto foi desenvolvido durante o evento **NLW (Next Level Week)** da **Rocketseat**, nas aulas da trilha Operator. O app explora tecnologias modernas de desenvolvimento web e integração com IA para criar uma experiência única de feedback de código.

## 🛠️ Tecnologias Principais

- **Next.js**: Framework React para o front-end e rotas de API.
- **Tailwind CSS 4.0**: Estilização rápida e moderna com sistema de temas baseado em variáveis nativas.
- **Shiki**: Syntax highlighting de alta qualidade para os blocos de código.
- **Biome**: Ferramenta rápida para linting e formatação de código, garantindo qualidade e consistência.
